## 📌 Project Description
This is a simple **Log Analyzer tool** built using Python.  
It is designed to simulate basic **cybersecurity incident detection (SOC analysis)** by reading server logs and identifying suspicious activities.

The tool helps analyze:
- IP addresses
- HTTP status codes
- Suspicious requests
- Possible attack patterns

---

## 🎯 Features
✔ Extract IP addresses from logs  
✔ Detect suspicious activity (login attempts, admin access, SQL keywords, errors)  
✔ Count HTTP response codes (200, 404, 500, etc.)  
✔ Identify most active / suspicious IP addresses  
✔ Generate simple security analysis output
